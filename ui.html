            Select frame → Export panel (right sidebar) → chọn PNG → drag file từ Figma vào dropzone


            Option C — Screenshot:

            macOS: Shift+Ctrl+Cmd+4 (clipboard) rồi paste

            Windows: Win+Shift+S rồi paste
